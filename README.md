Caso de Estudio: Árboles y Grafos en C# <br>
Contexto: <br>
El Parque Tecnológico “Innovatec” desea desarrollar un programa que permita:<br>
- Administrar la jerarquía organizativa del parque mediante un árbol.<br>
- Modelar las rutas internas entre edificios utilizando un grafo.<br>

<br>Parte A: Árbol — Jerarquía Organizativa <br>
Requerimientos:
- Registrar la jerarquía como un árbol general.
- Implementar funcionalidades de recorrido, búsqueda, conteo y niveles. 

<br>Parte B: Grafo — Sistema de Rutas del Parque <br>
Requerimientos:
- Representar edificios y conexiones mediante un grafo no dirigido y ponderado.
- Implementar funciones para mostrar conexiones, validar conexidad y calcular rutas. 

<br>Producto final esperado:
- Código funcional en C#.
- Informe con explicación del árbol y grafo implementados.
- Ruta más corta calculada.
